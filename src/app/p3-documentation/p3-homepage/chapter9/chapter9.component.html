9.1 Installation and Configuration

How do I install Plannotate on my iPad?

Please refer to Section 1.1 of the documentation for detailed steps on installing Plannoate on your iPad.

What versions of iOS does Plannoate support?

Plannotate supports iOS versions 13.x and higher. Note that previous versions of the iOS operating system my work with Plannotate but have not been fully tested.

9.2 Differences with iPad and Web Client

What are the main differences between the iPad and Web Client versions of Plannotate?

On a global level, the following differences will apply throughout the product.

  • On the web client, a left click on your mouse button is used as a single tap, and a right click on your mouse button is used for a long hold.
