Everything you need to be successful with your field inspections.

Plannotate version 3 ("P3", "Plannotate") is the successor to Plannotate version 2 ("P2", "Plannotate2"), a software platform for the collection of field investigation and condition assessment data. Plannotate is a SaaS application for the Apple iPad iOS and computer web browser that interfaces with an internet accessible web server, otherwise known as the Plannotate Cloud.

Plannotate is designed for the efficient collection of field data. As such, it allows powerful annotation capabilities for PDF documents and photos using various annotation icons or drawing tools chosen from a series of custom toolbars. Each annotation can then contain a variety of data such as forms or photos. The collected data can be synced to the cloud to provide data back-up, access to data for multiple users, and near real-time opportunities to review from anywhere in the world.

The user guide was drafted using the iPad interface and actions. In cases where the web client and iPad interface or actions differ, there will be an indication highlighted in green notating the differences. Please refer to Section 9.2 in the FAQs for detailed information.
