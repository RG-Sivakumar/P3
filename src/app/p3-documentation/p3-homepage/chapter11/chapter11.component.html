11.14 Plannotate 3.1.0

Version 3.1.0 - Released September 20, 2021

The 3.1.0 is a Major release and introduces a number of product enhancments, performance improvements, new features, and the first release of a full web client that mirrors the iPad release.

Note

With the initial release of Plannotate Web Client, the product documentation has been updated to reflect variances between the web client user interface and the iPad user interface. Note the following major differences in this release.

  • On the web client, a left click on your mouse button is used as a single tap, and a right click on your mouse button is used for a long hold.
  • Plannotate on web client communicates in real time with the Plannotate cloud and therefore does not have a manual synchronization feature.
  • Exporting to CSV, PDF and Photos have a new workflow. When a request is made on any of the export actions, a popup message indicating that you will receive the information in an email is displayed. The resulting email will have a link to download the information, which will be available for 24 hours. This was done as a performance enhancment, as projects with a large number of annotations can take significant time to generate exports.
  • A new sync status indicator has been implemented so that users can have better exposure to how far along a sync has progressed while running.
  • There is currently no undo or redo function on the web client. This is a feature that will be added in a future enhancement.
  • Resizing and rotating annotations on the web is done through a + and - button rather than draging motion.

Warning

On web client, when adding information to a form field on an annotation, you must hit Submit for the data to be saved. In addition, if you are changing a property on the annotation icon, you must hit Apply.

In a future release we will be removing the need to hit the submit button.

11.16 Plannotate 3.1.2

Version 3.1.2 - Released November 30, 2021

The 3.1.2 is a scheduled release and introduces a number of product enhancments, performance improvements, new features, and many fixes.

This release also has a number of fixes related to synchronization.

  1. There have been numerous special characters appearing in form fields that have come from either the use of an Apple Pencil, or by copy / paste of text where a special character was used. These have now been proactively identified and should not cause sync issues going forward.
  2. There have also been issues with special characters appearing in file names. Plannotate runs on Linux servers, and file naming restrictions vary from what is allowed on Windows and Mac computers. We have now built in support to handle these situation.

This release has additional performance improvements both on the web client and iPad.

  1. The web has been better optimized to handle documents with a large volume of annotations so that load times will improve dramatically.
  2. The time delay when switching pages on the web has also been improved.
  3. Load times on the iPad when opening documents, loading annotations, photos and form data is now significally faster.
  4. Rendering annotations on documents within the iPad has been improvied.
  5. Memory issues have been optimized on the iPad to avoid memory leaks that were causing crashes.
  6. A lazy loading technique has been implemented so that images and data will load in the background without slowing down other operations.
