Sign Out of Plannotate

Plannotate allows users to work offline and retains work performed on each device that has not yet been synchronized to the Plannotate cloud server. Once a manual or automated sync takes place, the information will be synchronized with the Plannotate cloud.

Upon signing out from this device, you have the option of removing any of your work that has not yet been synchronized. Note that if selected, this operation cannot be undone.

If you wish to remove all work that has not been synchronized, please select the checkbox below. Otherwise local work will be retained on the device.
