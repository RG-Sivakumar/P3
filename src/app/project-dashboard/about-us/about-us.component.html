Plannotate3 ("P3") is the successor to Plannotate2 ("P2"), a software platform for the collection of field investigation and condition assessment data. P3 is a SaaS application for the Apple iPad iOS and computer web browser that interfaces with an internet accessible web server, otherwise known as the P3 Cloud.
P3 is designed for the efficient collection of field data. As such, it allows powerful annotation capabilities for PDF documents and photos using various annotation icons or drawing tools chosen from a series of custom toolbars. Each annotation can then contain a plethora of data such as forms or photos. The collected data can be synced to the cloud to provide data back-up, access to data for multiple users, and near real-time opportunities to review from anywhere in the world.
